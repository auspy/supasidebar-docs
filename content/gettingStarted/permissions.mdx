# Setting Up Permissions

SupaSidebar requires **Automation** and **Accessibility** permissions for browser integration and shortcuts.

## Automation

Automation permission allows SupaSidebar to interact with your browsers to save tabs, copy URLs, and detect the current page.

1. Go to **System Settings → Privacy & Security → Automation**
2. Find **SupaSidebar** and enable checkboxes for all browsers you use

![Automation Permission](/assets/automation%20real.jpeg)

When you first use browser shortcuts, macOS will prompt for permission. Click **OK** to grant access.

## Accessibility

Accessibility permission is required for copying text, mouse triggers, and interacting with browsers.

1. Go to **System Settings → Privacy & Security → Accessibility**
2. Find **SupaSidebar** and enable the checkbox

![Accessibility](/assets/accessibility%20mouse%20triggers.jpeg)
