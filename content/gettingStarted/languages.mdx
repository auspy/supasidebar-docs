# Languages

SupaSidebar follows your Mac's system language by default. You can override the language in **Preferences - General**.

## Available languages

SupaSidebar is fully translated into 11 languages beyond the English source. Every supported language has complete coverage of the interface.

| Language | Coverage |
|---------|----------|
| English | 100% (source) |
| French | 100% |
| German | 100% |
| Spanish | 100% |
| Italian | 100% |
| Japanese | 100% |
| Korean | 100% |
| Dutch | 100% |
| Brazilian Portuguese | 100% |
| Russian | 100% |
| Simplified Chinese | 100% |
| Traditional Chinese | 100% |

Translations are refined with each release. If a phrase reads awkwardly in your language, see the reporting section below.

## Changing the language

1. Open **Preferences - General**
2. Find the **Language** picker
3. Pick a language and restart SupaSidebar

If your preferred language isn't listed, SupaSidebar falls back to English.

## Reporting translation issues

If you spot a translation that reads awkwardly or is plain wrong, please let us know - the easiest way is to email support with a screenshot. Translation fixes ship in regular releases alongside other updates.
