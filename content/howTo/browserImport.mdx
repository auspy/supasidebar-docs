# Import Bookmarks

Bring your bookmarks from Arc or Chrome into SupaSidebar.

## Import Steps

1. Open **Preferences** â†’ **Import**
2. Click **Import from Arc** or **Import from Chrome**
3. Grant Automation permission if prompted
4. Select what to import
5. Click **Start Import**

## What Gets Imported

| Browser | Imports |
|---------|---------|
| **Arc** | Spaces, folders, pinned tabs, bookmarks |
| **Chrome** | Bookmarks Bar, Other Bookmarks, folder hierarchy |

## Other Browsers

For Safari, Firefox, Edge, or other browsers: first import your bookmarks into Chrome or Arc, then import from there into SupaSidebar.

*This is a temporary workaround. Direct import for other browsers is coming soon.*
