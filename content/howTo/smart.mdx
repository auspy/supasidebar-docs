# Smart Shortcuts

Context-aware shortcuts that adapt to your active application.

![Smart Save](/assets/smart%20save.jpeg)

## Overview

| Shortcut | Browser | Finder |
|----------|---------|--------|
| `Cmd Ctrl S` | Save current page | Add selected items |
| `Cmd Ctrl C` | Copy page URL | Copy file paths |

## Smart Save (`Cmd Ctrl S`)

### In Browser
1. Navigate to a page you want to save
2. Press `Cmd Ctrl S`
3. Page title and URL are saved to your sidebar

### In Finder
1. Select one or more files/folders
2. Press `Cmd Ctrl S`
3. Items are added to your sidebar

## Smart Copy (`Cmd Ctrl C`)

### In Browser
1. Navigate to the page you want to share
2. Press `Cmd Ctrl C`
3. URL is copied to clipboard, ready to paste

### In Finder
1. Select one or more files/folders
2. Press `Cmd Ctrl C`
3. File paths are copied (one per line for multiple items)

## Supported Browsers

Safari, Chrome, Firefox, Edge, Arc, Brave, Vivaldi, Comet, Orion, Atlas, Zen, Wavebox, Dia, and Helium.

## Permissions

Smart Shortcuts require Automation permissions. Grant access in **System Settings → Privacy & Security → Automation → SupaSidebar**.
