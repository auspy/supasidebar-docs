# Drag and Drop

Add files, folders, and links to your sidebar by dragging them directly into the panel.

## From Finder

1. Select files or folders in Finder
2. Drag them to the SupaSidebar panel
3. Drop to add immediately

![Folder getting saved on sidebar](/assets/Folder%20getting%20save%20on%20sidebar.jpeg)

## Tips

- Drag multiple items at once for batch adding
- Drop into a folder to organize immediately
- Works with any file type
