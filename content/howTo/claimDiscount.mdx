# Claim a Discount Code

Got a discount code? Here's how to apply it during checkout and save on your SupaSidebar purchase.

## Where to Get Discount Codes

You can get discount codes through:

- **Seasonal promotions**  - announced on our website and social media
- **Student discount**  - verify your student email at [supasidebar.com/student](https://supasidebar.com/student)
- **Content creators**  - share about SupaSidebar and earn a discount at [supasidebar.com/promote](https://supasidebar.com/promote)
- **Referral program**  - get a code when a friend refers you via [supasidebar.com/gift](https://supasidebar.com/gift)

## How to Apply Your Discount Code

### Step 1: Open the Pricing Page

Go to [supasidebar.com/pricing](https://supasidebar.com/pricing) in your browser.

{/* Screenshot: Pricing page */}

### Step 2: Choose a Plan

Select the plan that works best for you  - Monthly, Yearly, or Lifetime.

{/* Screenshot: Plan selection */}

### Step 3: Enter Your Discount Code

In the checkout overlay, look for the **discount code** field. Type or paste your code into the field.

{/* Screenshot: Discount code input field */}

### Step 4: Validate the Code

Click the **Validate** button next to the code field. The discount will be validated and your updated price will be shown immediately.

{/* Screenshot: Code applied, showing discounted price */}

### Step 5: Complete Your Purchase

Review the discounted total and complete the payment. That's it  - enjoy SupaSidebar at a discounted price!

{/* Screenshot: Final checkout with discount applied */}

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Invalid code" error | Double-check the code for typos. Codes are case-sensitive. |
| "Code expired" error | The promotion period has ended. Check our social media for active promotions. |
| Code doesn't apply to your plan | Some codes only work with specific plans (e.g., subscriptions only). |
| Discount not showing | Make sure you clicked **Validate** after entering the code. |

## Need Help?

If your discount code isn't working, reach out to us at [supasidebar.com/contact](https://supasidebar.com/contact) and we'll sort it out.
