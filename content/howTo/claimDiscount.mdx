# Claim a Discount Code

Got a discount code? Here's how to apply it during checkout and save on your SupaSidebar purchase.

## Where to Get Discount Codes

You can get discount codes through:

- **Seasonal promotions** — announced on our website and social media
- **Student discount** — verify your student email at [supersidebar.com/student](https://supersidebar.com/student)
- **Content creators** — share about SupaSidebar and earn a discount at [supersidebar.com/promote](https://supersidebar.com/promote)
- **Referral program** — get a code when a friend refers you via [supersidebar.com/gift](https://supersidebar.com/gift)

## How to Apply Your Discount Code

### Step 1: Open the Pricing Page

Go to [supersidebar.com/pricing](https://supersidebar.com/pricing) in your browser.

{/* Screenshot: Pricing page */}

### Step 2: Choose a Plan

Select the plan that works best for you — Monthly, Yearly, One Year, or Lifetime.

{/* Screenshot: Plan selection */}

### Step 3: Click "Get Started" or "Buy Now"

Click the purchase button on your chosen plan. This will open the checkout overlay.

{/* Screenshot: Checkout overlay opening */}

### Step 4: Enter Your Discount Code

In the checkout overlay, look for the **discount code** field. Type or paste your code into the field.

{/* Screenshot: Discount code input field */}

### Step 5: Validate the Code

Click the **Apply** button next to the code field. The discount will be validated and your updated price will be shown immediately.

{/* Screenshot: Code applied, showing discounted price */}

### Step 6: Complete Your Purchase

Review the discounted total and complete the payment. That's it — enjoy SupaSidebar at a discounted price!

{/* Screenshot: Final checkout with discount applied */}

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Invalid code" error | Double-check the code for typos. Codes are case-sensitive. |
| "Code expired" error | The promotion period has ended. Check our social media for active promotions. |
| Code doesn't apply to your plan | Some codes only work with specific plans (e.g., subscriptions only). |
| Discount not showing | Make sure you clicked **Apply** after entering the code. |

## Need Help?

If your discount code isn't working, reach out to us at [supersidebar.com/contact](https://supersidebar.com/contact) and we'll sort it out.
