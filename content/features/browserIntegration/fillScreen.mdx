# Fill Screen

Browser automatically resizes to fill the remaining screen space.

## How It Works

{/* ![Fill Screen Mode](/assets/browser-integration/fill-screen.jpeg) */}

When the sidebar opens, your browser shrinks to fit the remaining space. When closed, the browser expands back to fill the screen.

## Setup

1. Open **Preferences → General**
2. Set **Browser Integration** to **Smart Attach**
3. Enable **Fill Entire Screen**
4. Grant Accessibility permissions when prompted

## When Sidebar Hides

By default, browser windows expand to fill the full screen when the sidebar hides. You can change this behavior in **Preferences → General → When Sidebar Hides**. See [Sidebar Panel  - Behavior](/features/sidebar#behavior) for all options.

## When to Use

- You want a split-screen layout
- You always use sidebar and browser together
- You prefer maximized use of screen space
