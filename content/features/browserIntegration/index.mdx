# Browser Integration

Choose how SupaSidebar interacts with your browser windows.

Configure in **Preferences → General → Browser Integration**.

## Modes

| Mode | Description |
|------|-------------|
| **Independent** | Sidebar opens on the screen edge with no browser interaction |
| **Smart Attach** | Sidebar attaches to your browser and manages window layout |

## Smart Attach

Smart Attach is the recommended mode. When enabled, the sidebar intelligently positions itself next to your browser and adjusts windows to avoid overlap. It includes several features you can enable in Preferences:

| Feature | Description |
|---------|-------------|
| [Fill Entire Screen](/features/browserIntegration/fillScreen) | Browser resizes to fill the remaining space alongside the sidebar |
| [Window Tiling](/features/browserIntegration/tiling) | Auto-tiles 2-3 browser windows as equal-width columns |
| [Prevent Browser Overlap](/features/browserIntegration/smart) | Automatically moves browser windows out from under the sidebar |
| Auto-Switch Side | Moves sidebar to the other side when it would cover another app |
| [When Sidebar Hides](/features/sidebar#behavior) | Choose whether windows fill the remaining space or restore to their original position |

## Independent

The sidebar opens on the screen edge without affecting browser windows. Use this if you prefer full manual control over your window layout.

## Profile-Aware Behavior

When a space is linked to a browser profile, Smart Attach automatically targets the profile's browser window. Switching spaces re-applies the integration to the new profile's browser.

See [Browser Profiles](/features/profileSpaces) for details.

## Sidebar Hide Behavior

When using Smart Attach, the sidebar's auto-hide behavior in browsers is controlled separately from general [mouse trigger](/features/sidebarAccess/mouseTriggers) settings. In non-browser apps, the sidebar automatically hides when you move your mouse away. In browsers, the sidebar may stay visible because Smart Attach is actively managing your window layout.

To configure how the sidebar hides when a browser is active, go to **Preferences → Sidebar → Sidebar Behavior** and enable the browser auto-hide setting. There may be a short delay before auto-hide activates — this prevents the sidebar from flickering during normal use.

If the sidebar isn't auto-hiding in browsers, see [Sidebar Doesn't Hide in Browser](/troubleshooting/sidebarDoesntHideInBrowser).

## Permissions

Smart Attach requires Accessibility permissions. See [Permissions Guide](/gettingStarted/permissions).
