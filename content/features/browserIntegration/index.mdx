# Browser Integration

Choose how SupaSidebar interacts with your browser windows.

Configure in **Preferences → General → Browser Integration**.

## Modes

| Mode | Description |
|------|-------------|
| [Attach to Browser](/features/browserIntegration/attach) | Sidebar floats alongside browser |
| [Smart Positioning](/features/browserIntegration/smart) | Moves overlapping windows |
| [Window Tiling](/features/browserIntegration/tiling) | Auto-tiles 2-3 windows as columns |
| [Fill Screen](/features/browserIntegration/fillScreen) | Browser resizes to fill space |
| [Dock Mode](/features/browserIntegration/dock) | Browser maximizes when sidebar closes |

**None** mode runs sidebar independently with no window adjustments.

## Profile-Aware Behavior

When a space is linked to a browser profile, all integration modes (Attach, Smart, Fill Screen, Dock) automatically target the profile's browser window. Switching spaces re-applies the integration to the new profile's browser.

See [Browser Profiles](/features/profileSpaces) for details.

## Permissions

All modes except None require Accessibility permissions. See [Permissions Guide](/gettingStarted/permissions).
