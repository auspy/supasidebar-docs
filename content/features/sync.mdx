# iCloud Sync

Keep your links, folders, spaces, and tags synchronized across all your Macs using iCloud.

## What Syncs

| Data | Synced |
|------|--------|
| Links | All saved websites, files, and folders |
| Folders | Your folder structure and organization |
| Spaces | Workspace configurations and assignments |
| Tags | Tag names, colors, and descriptions |
| Settings | Most preferences (excluding device-specific options) |

## Setup

iCloud sync is enabled by default when signed into iCloud on your Mac.

### Enable iCloud for SupaSidebar

If sync isn't working:

1. Open **System Settings → Apple ID → iCloud**
2. Scroll to **Apps Using iCloud**
3. Ensure **SupaSidebar** is enabled
