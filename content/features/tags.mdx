# Tags & AI Tagging

Organize your links with color-coded tags and let AI automatically categorize new additions.

## How It Works

Tags are labels you attach to links for quick filtering and organization. Each tag has a name, color, and optional description. You can assign tags manually or let AI do it automatically based on link content.

### AI Tagging Providers

SupaSidebar supports three tagging modes:

- **Manual/Keyword**: You assign tags yourself
- **OpenAI Cloud**: Uses GPT models for intelligent categorization
- **Local AI (MLX)**: Runs on-device using Apple Silicon - no API key needed

## Setup

### Creating Tags

1. Open **Preferences** → **Tags** tab
2. Click the **+** button to add a new tag
3. Enter a name (e.g., "Work", "Research", "Read Later")
4. Choose a color
5. Optionally add a description to help AI understand the tag's purpose

### Configuring AI Tagging

1. Open **Preferences** → **Tags** tab
2. Under **AI Provider**, select your preferred option:
   - **Manual**: No AI assistance
   - **OpenAI**: Enter your API key when prompted
   - **Local AI**: Click **Download Model** to install the MLX model
3. Enable **Auto-tag new links** to automatically tag links when saved

## Using Tags

### Assigning Tags Manually

- **Single link**: Right-click a link → **Edit Tags** → Select tags from the picker
- **Multiple links**: Select links → Right-click → **Edit Tags** → Apply to all

### Bulk AI Tagging

1. Select the links you want to tag
2. Right-click → **Apply AI Tags**
3. AI analyzes each link and suggests matching tags
4. Review and confirm the suggestions

### Filtering by Tags

- Click a tag in the sidebar to filter links
- Use the **Command Panel** (⌘⌃K) and type a tag name
- Combine tags with Smart Folders for automatic organization

## AI Tag Descriptions

Better tag descriptions lead to better AI matching. Compare:

| Poor Description | Better Description |
|------------------|-------------------|
| "Work stuff" | "Links related to my job at Acme Corp, including project docs, internal tools, and team resources" |
| "Articles" | "Long-form articles, blog posts, and essays I want to read - not news or reference docs" |

### Generating Descriptions with AI

1. Select a tag in Preferences
2. Click **Generate Description**
3. AI creates a description based on links already tagged
4. Edit as needed

## Troubleshooting

**AI tags are inaccurate**
- Add more descriptive tag descriptions
- Ensure tags have enough example links for AI to learn from
- Check that your OpenAI API key is valid (if using cloud)

**Local AI model won't download**
- Requires macOS 14+ and Apple Silicon
- Check internet connection
- Ensure sufficient disk space (~500MB)

**Tags not syncing across devices**
- Verify iCloud sync is enabled in Preferences
- Check iCloud account status in System Settings
