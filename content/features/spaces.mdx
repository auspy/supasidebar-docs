# Spaces

Spaces in SupaSidebar let you organize your links, files, and folders into separate workspaces - switch between work, personal, and projects instantly.

<Callout type="info">
Spaces is a Premium feature. Free users can use up to 3 spaces. [Upgrade](https://supasidebar.com/pricing) for unlimited spaces.
</Callout>

![Saved Spaces](/assets/saved%20spaces.jpeg)

## Features

- **Independent Workspaces** - Each space has its own links, folders, and smart folders
- **Instant Switching** - Jump between spaces with keyboard shortcuts
- **Color Coding** - Assign colors for visual distinction
- **Custom Icons** - Set SF Symbols, emoji, or upload a custom image as the space icon
- **Filtered Live Tabs** - Browser tabs filtered by current space
- **Shared Pinned Items** - Pinned items accessible across all spaces

## Browser Profiles

Link a browser profile to any space for complete session isolation. Each profile has its own cookies, history, and extensions  - switching spaces instantly swaps which profile is visible.

See [Browser Profiles](/features/profileSpaces) for setup and usage.

## Settings

Configure spaces in **Preferences → Spaces**.

### Space Management

- **Default startup space** - Choose which space opens on launch
- **Space navigation** - Enable keyboard shortcuts for switching
- **Number shortcuts** - Use `Cmd Ctrl 1-9` to jump to specific spaces
- **Reorder spaces** - Drag spaces to rearrange their order

![Default setup and new space](/assets/automation.jpeg)

## Usage

- **Create a space** - Click space selector → + New Space → Name and set color
- **Save to space** - Links saved with `Cmd Ctrl S` go to current space
- **Move between spaces** - Right-click item → Move to Space
- **Organize** - Create folders and smart folders within each space

![Space and folders](/assets/space%20and%20folders.jpeg)

## Keyboard Shortcuts

Shortcuts specific to spaces. For all shortcuts, see [Shortcuts](/features/shortcuts).

| Shortcut | Action |
|----------|--------|
| `Cmd Ctrl ←` | Previous space |
| `Cmd Ctrl →` | Next space |
| `Cmd Ctrl 1-9` | Jump to space 1-9 (if enabled) |
