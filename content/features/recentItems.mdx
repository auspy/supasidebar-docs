# Recent Links

Automatic tracking of your browsing history across all supported browsers - recently visited pages appear without manual saving.

![Recent Items](/assets/recent%20itmes.jpeg)

## Features

- **Auto-Tracking** - Pages you visit are automatically captured
- **Multi-Browser** - Tracks Safari, Chrome, Edge, Brave, Firefox, and more
- **30-Day History** - Recent links persist for 30 days
- **Available in every space** - Recently visited links appear in all your spaces, not only the space you were in when you opened them
- **Searchable** - Find recent links via Command Panel (`Cmd Ctrl K`)
- **Privacy Blocklist** - Blocked sites and browsers are excluded from recent items. See [Privacy Blocklist](/features/privacyBlocklist)

## Settings

Configure recent links in **Preferences → Recents**.

- **Mode** - Choose Recent Links or [Live Tabs](/features/liveTabs)
- **Display limit** - Number of recent links to show (default: 30)
- **Background tab sync** - Opens existing tab instead of creating duplicates
- **Clear recents** - Remove all recent links history

### Permissions

Browser integration requires Automation permissions. See [Setup](/gettingStarted/SetUp) for details.

## View-mode switch

The Recent Links section header has a toggle that switches the list between **compact** and **detailed** views. Compact shows favicon and title on one line, detailed adds the URL. The same toggle is on the [Live Tabs](/features/liveTabs) header, and the choice is remembered per section.

## Recent Links vs Live Tabs

| Mode | Shows | Persists |
|------|-------|----------|
| Recent Links | Visited pages (30 days) | After browser closes |
| Live Tabs | Currently open tabs | Only while open |

## Keyboard Shortcuts

Shortcuts specific to recent links. For all shortcuts, see [Shortcuts](/features/shortcuts).

| Shortcut | Action |
|----------|--------|
| `Cmd Ctrl K` | Open Command Panel to search recents |
| `Return` | Open selected link |
