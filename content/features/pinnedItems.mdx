# Pinned Links

Pinned Links in SupaSidebar give you instant access to your most important links, files, and apps with single-keystroke shortcuts.

![Pinned Links](/assets/Pinned%20Links.jpeg)

## Features

- **Top of Sidebar** - Pinned links always appear at the top with a pin icon
- **Keyboard Shortcuts** - Open pins 1-9 with `Cmd Opt 1-9` from anywhere
- **Cross-Space Access** - Pinned links visible across all spaces
- **Custom Browser** - Set preferred browser per pinned link
- **Custom Icons** - Set SF Symbols, emoji, or upload a custom image as the pin icon
- **Drag to Reorder** - Drag pins to rearrange them; position determines shortcut number
- **Smart Tab Reuse** - Clicking a pin activates the existing browser tab if the URL is already open, instead of opening a duplicate

## Settings

Configure pinned links in **Preferences → General**.

### Shortcuts

- **Pinned link number shortcuts** - Enable `Cmd Opt 1-9` to open pins

## Usage

- **Pin a link** - Right-click → Pin to Top, or select and press `Cmd P`
- **Reorder pins** - Drag to change position (position = shortcut number)
- **Unpin** - Right-click → Unpin, or select and press `Cmd P`
- **Set browser** - Right-click → Set Browser
- **Change icon** - Right-click → Edit Icon to choose an SF Symbol, emoji, or upload a custom image

## Keyboard Shortcuts

Shortcuts specific to pinned links. For all shortcuts, see [Shortcuts](/features/shortcuts).

| Shortcut | Action |
|----------|--------|
| `Cmd Opt 1-9` | Open pinned link 1-9 |
| `Cmd P` | Pin or unpin selected link |
