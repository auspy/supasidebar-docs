# Global Shortcuts

System-wide keyboard shortcuts for instant access to SupaSidebar from any application - no mouse required.

### Smart Shortcuts (Context-Aware)

**⌘⌃S** and **⌘⌃C** are context-aware shortcuts that adapt based on your active application:

| Shortcut | Action | Description |
|----------|--------|-------------|
| **⌘⌃S** | Smart Save | Save browser page OR add Finder items (context-aware) |
| **⌘⌃C** | Smart Copy | Copy URL OR file paths to clipboard (context-aware) |
| **⌘⌥A** | Add Folder | Add current Finder folder (ignores selection) |

**Learn more:** [Smart Shortcuts →](smartShortcuts) - Deep dive into context-aware behavior across browsers and Finder

### Space Navigation

| Shortcut | Action | Description |

| **⌘⌃→** | Next Space | Navigate to next desktop space |

## Customizing Shortcuts

### Changing Shortcuts

1. Open **Preferences → Shortcuts**
2. Click on any shortcut row
3. Press your desired key combination
4. New shortcut is saved automatically



### Shortcut Requirements

Valid shortcuts must include at least one modifier key:
- **⌘** Command (required for most)
- **⌃** Control
- **⌥** Option
- **⇧** Shift

**Recommended Patterns:**
- **⌘⌃** + key (Command + Control)
- **⌘⇧** + key (Command + Shift)
- **⌘⌥** + key (Command + Option)

Avoid single-key or simple combinations that might conflict with other apps.

### Resolving Conflicts

If a shortcut conflicts with another application:

**Option 1: Change SupaSidebar Shortcut**
1. Open Preferences → Shortcuts
2. Click conflicting shortcut
3. Record new combination

**Option 2: Disable Conflicting App**
- Check other app's preferences
- Disable or change their shortcut
- SupaSidebar shortcut will work

**Option 3: Disable Optional Shortcuts**
- Space navigation number shortcuts
- Pinned item shortcuts
- Only keep essential shortcuts active

## Troubleshooting

**Shortcut not working?**
- Check for conflicts with other apps
- Try disabling other app's shortcut temporarily
- Change SupaSidebar shortcut in Preferences
- Restart SupaSidebar

**Shortcuts work inconsistently?**
- Some apps may capture keyboard events first
- Try different modifier combinations
- Report specific app conflicts to support

**Browser/Finder shortcuts failing?**
- Verify automation permissions granted
- System Settings → Automation → Check SupaSidebar
- Restart relevant application

**Space/Pinned shortcuts missing?**
- These are optional - enable in Preferences
- Preferences → Spaces → Enable navigation shortcuts
- Preferences → General → Enable pinned item shortcuts

**Forgot a shortcut?**
- Open Preferences → Shortcuts to see all
- Or check this documentation page
- Most essential: ⌘⇧Space (toggle sidebar)

## Accessibility

### Keyboard-Only Operation

SupaSidebar is fully keyboard-accessible:
- Every feature has a keyboard shortcut or arrow key navigation
- No mouse required for any operation
- Perfect for users with motor disabilities or efficiency preferences

### VoiceOver Support

Screen reader users can:
- Navigate sidebar with VoiceOver commands
- Hear link titles and URLs
- Use all keyboard shortcuts normally

### Reduced Motion

Sidebar animations respect macOS **Reduce Motion** accessibility setting.
