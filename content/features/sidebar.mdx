# Sidebar Panel

The SupaSidebar panel is a floating sidebar for macOS that provides instant access to your saved links, files, and folders from any application.

![Sidebar Panel position on macOS](/assets/default%20position%20of%20sidebar.jpeg)

## Features

- **Multi-Screen Support** - Automatically appears on your active display
- **Left or Right Positioning** - Toggle sides instantly with `Cmd Ctrl L`
- **Auto-hide** - Optionally hides when clicking outside the panel
- **Resizable** - Drag to adjust panel width
- **Pin to Spaces** - Follow across all macOS desktop spaces or stay on one

## Settings

Configure the sidebar panel in **Preferences**.

### Appearance

Found in Preferences → Appearance.

- **Width** - Drag the sidebar edge to resize
- **Opacity on focus change** - Dim sidebar when not focused
- **Glass theme** - Enable translucent background
- **Show item counts** - Display count badges on folders
- **Arrow indicator** - Show expand/collapse arrows on folders

### Behavior

Found in Preferences → General.

- **Auto-hide on deactivate** - Hide sidebar when clicking outside
- **Default side** - Choose left or right screen edge
- **Auto show with browser** - Show sidebar when browser is active. See [Live Tabs](/features/liveTabs)
- **Adaptive height** - Adjust sidebar height based on content
- **Browser integration** - Configure how sidebar interacts with browsers. See [Browser Integration](/features/browserIntegration)
- **When Sidebar Hides** - Control what happens to browser windows when the sidebar is hidden:
  - **Fill Remaining Space** (default)  - browser windows expand to fill the full screen. Tiled windows re-tile across the entire screen width; filled windows take up the whole screen. Only applies when [Fill Entire Screen](/features/browserIntegration/fillScreen) or [Auto-Tile](/features/browserIntegration/tiling) is active.
  - **Restore Original Position**  - browser windows return to their size and position from before the sidebar was opened.

## Keyboard Shortcuts

Shortcuts specific to the sidebar panel. For all shortcuts, see [Shortcuts](/features/shortcuts).

| Shortcut | Action |
|----------|--------|
| `Cmd Shift Space` | Show or hide sidebar |
| `Cmd Ctrl F` | Bring sidebar to front |
| `Cmd Ctrl L` | Switch between left and right |
| `Cmd Ctrl P` | Pin or unpin from current space |
| `Esc` | Hide sidebar |
