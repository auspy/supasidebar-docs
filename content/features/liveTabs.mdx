# Live Tabs

Real-time mirror of your currently open browser tabs directly in the sidebar - switch between tabs across all browsers without leaving your keyboard.

## How It Works


Live Tabs continuously monitors your supported browsers and displays all open tabs in real-time within the sidebar. Each tab shows its title, URL, and browser icon.

### Real-Time Synchronization

**Accessibility Observer** 
Instant updates using macOS Accessibility API when tabs are created, switched, or closed - no polling delay.

## Multi-Browser Support

Live Tabs works across all supported browsers simultaneously:

**Fully Supported** (Real-time tracking):Safari, Chrome, Edge, Brave, Vivaldi, Comet, Orion, Atlas (ChatGPT), Helium

**In Development**:Firefox, Arc, Dia


![Live Tabs in Multiple Browsers](/assets/lifetabsmultiplebrowser.jpeg)

### Browser Order Customization

Organize which browsers appear first in your live tabs list:

1. Open **Preferences → Live Tabs**
2. Drag browsers to reorder priority
3. Your most-used browser's tabs will appear first

## Setup

### Enable Live Tabs

1. Open **Preferences → Recents**
2. Toggle **Live Tabs** mode
3. Grant browser permissions when prompted



![Live Tabs Option](/assets/Live%20Tabs%20Option.jpeg)

## Using Live Tabs

### View All Tabs

Press **⌘⌃K** to open the Command Panel - your live tabs appear alongside saved links and recent items.

In the sidebar, live tabs appear in the **Recents** section (when Live Tabs mode is enabled).

### Switch to a Tab

**From Sidebar:**
1. Navigate to the tab with arrow keys or click
2. Press **Return** or click to activate

**From Command Panel:**
1. Press **⌘⌃K**
2. Type tab title or URL
3. Press **Return** to switch

## Troubleshooting

**Tabs not appearing?**
- Verify automation permissions are granted in System Settings
- Ensure Live Tabs mode is enabled in Preferences → Recents
- Check that your browser is in the supported list

**Tabs updating slowly?**
- Grant Accessibility permissions for real-time updates (Direct Distribution only)
- Close unused browsers to improve performance

**Tabs from wrong space showing?**
- Enable "Filter by current space" in Preferences → Live Tabs
- Ensure websites are saved to the correct space
