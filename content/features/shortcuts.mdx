# Shortcuts

![Global Shortcuts](/assets/Global%20Shortcuts.jpeg)

## Sidebar Control

| Shortcut | Action |
|----------|--------|
| `Cmd Shift Space` | Toggle Sidebar |
| `Cmd Ctrl K` | Open Command Panel |
| `Cmd Ctrl F` | Focus Sidebar |
| `Cmd Ctrl P` | Pin Sidebar (follow across spaces) |
| `Cmd Ctrl L` | Toggle Side (left/right) |

## Smart Shortcuts

Context-aware shortcuts. See [Smart Shortcuts](/howTo/smart) for details.

| Shortcut | Action |
|----------|--------|
| `Cmd Ctrl S` | Save page (browser) or add items (Finder) |
| `Cmd Ctrl C` | Copy URL (browser) or file paths (Finder) |

## Browser Shortcuts

Shortcuts that interact with browsers require Accessibility permission. If a shortcut doesn't work, check **System Settings - Privacy & Security - Accessibility** and make sure SupaSidebar is enabled. See [Permissions](/gettingStarted/permissions) for setup.

| Shortcut | Action |
|----------|--------|
| `Cmd Ctrl T` | Save all browser tabs to folder |
| `Cmd Ctrl B` | Open current tab in another browser |

## Finder Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd Opt A` | Add current Finder folder |

## Space Navigation

Enable in **Preferences - Spaces**.

| Shortcut | Action |
|----------|--------|
| `Cmd Ctrl Left` | Previous Space |
| `Cmd Ctrl Right` | Next Space |
| `Cmd Ctrl 1-9` | Jump to Space 1-9 |

## Pinned Links

Enable in **Preferences - General**.

| Shortcut | Action |
|----------|--------|
| `Cmd Opt 1-9` | Open Pinned Link 1-9 |

## In-App Shortcuts

When sidebar is focused:

| Shortcut | Action |
|----------|--------|
| `Return` | Open selected link |
| `Space` | Edit link name |
| `Cmd C` | Copy URL |
| `Cmd V` | Paste URL as new link |
| `Cmd P` | Pin/Unpin link |
| `Cmd N` | Create new link |
| `Cmd R` | Rename link |
| `Cmd Delete` | Delete link |
| `Cmd [` | Compact sidebar (icon-only) |
| `Cmd ]` | Expand sidebar (full width) |
| `Escape` | Close sidebar |
| `Up` `Down` | Navigate links |
| `Left` `Right` | Collapse/Expand folders |
| `Up` `Down` (in smart folder) | Navigate items within smart folder |
| `Shift Up` `Shift Down` | Extend selection |

## Mouse Shortcuts

| Action | Result |
|--------|--------|
| `Click` | Open link |
| `Cmd Click` | Preview in mini window (web) / Toggle selection (files) |
| `Opt Click` | Choose browser to open link |
| `Shift Click` | Range select (from last selected) |

## Command Panel

When Command Panel is open (`Cmd Ctrl K`):

| Shortcut | Action |
|----------|--------|
| `Up` `Down` | Navigate results |
| `Return` | Open selected result |
| `Opt Return` | Actions panel (choose browser, reveal in sidebar/Finder) |
| `Shift Tab` | Cycle search scope (All, Recents, Live Tabs, Saved, Spaces, Folders, Settings) |
| `Ctrl Return` | Activate [Ask AI](/features/askAI) mode |
| `Cmd Return` | Search the web directly |
| `Escape` | Close panel |

## Settings

Customize shortcuts in **Preferences - Shortcuts**.

- Click any shortcut to change it
- Shortcuts require at least one modifier (`Cmd`, `Ctrl`, `Opt`, `Shift`)
- Click "Reset to Defaults" to restore original shortcuts
