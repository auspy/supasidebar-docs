## Enable Mouse Triggers

1. Open **Preferences → Mouse Triggers**
2. Toggle **Enable mouse triggers** ON
3. Select edge zones (left, right, or both)

![Mouse Trigger Settings](/assets/mouse%20trigger%20settings%20panel.jpeg)

Requires [Accessibility permission](/gettingStarted/permissions#accessibility): **System Settings → Privacy & Security → Accessibility → SupaSidebar**

## Settings

| Setting | Description |
|---------|-------------|
| Edge Threshold | How close to edge before triggering (smaller = closer) |
| Hover Delay | Time to hover before sidebar appears (increase if triggering accidentally) |
| Enabled Zones | Left edge, right edge, or both |

![Mouse Triggers](/assets/mouse%20triggers.jpeg)

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Not working | Grant Accessibility permission, restart app |
| Triggers too easily | Increase hover delay, decrease edge threshold |
| Not appearing | Check triggers enabled, verify edge zone is on |
| Multi-monitor issues | Test each display edge, check Dock isn't blocking |
