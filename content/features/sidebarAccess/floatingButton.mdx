## Enable Floating Button

1. Open **Preferences â†’ General**
2. Enable **Show floating button**
3. Button appears immediately

![Floating Button Toggle](/assets/Floating%20Button%20Toggle.jpeg)

## Features

- **Always visible** - Floats above all applications and desktop spaces
- **Draggable** - Drag to any position; persists across restarts
- **Multi-display** - Works across multiple monitors
- **Single click** - Toggle sidebar on/off

![Floating Button](/assets/floating%20button.jpeg)

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Button not appearing | Toggle off/on in Preferences, restart app |
| Wrong position | Drag to new location (saves automatically) |
| Button off-screen | Disable and re-enable to reset position |

Keyboard alternative: `Cmd Shift Space` provides the same toggle behavior.
