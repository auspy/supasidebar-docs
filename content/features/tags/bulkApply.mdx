# Bulk Apply Tags

Use AI to automatically find and tag existing links that match a tag's description - organize your entire library in seconds.

{/* TODO: Add screenshot - bulk-apply-overview.jpeg */}

## How It Works

1. Create a tag with an AI description
2. Click "Apply" to scan your existing links
3. AI finds matching links and shows confidence scores
4. Review and apply tags to selected links

## Step by Step

### 1. Open AI Tags Preferences

Go to **Preferences â†’ AI Tags** to see your tag definitions.

{/* TODO: Add screenshot - ai-tags-preferences.jpeg */}

### 2. Ensure Tag Has AI Description

Tags need an AI description for bulk apply to work. Click a tag to add or edit its description.

{/* TODO: Add screenshot - tag-ai-description.jpeg */}

### 3. Click Apply

Click the **Apply** button next to any tag with an AI description.

{/* TODO: Add screenshot - bulk-apply-button.jpeg */}

### 4. Review Matches

AI analyzes your links and shows matches with confidence scores:
- **Green** - High confidence (80% and above)
- **Orange** - Medium confidence (50-79%)
- **Gray** - Low confidence (under 50%)

High-confidence matches are auto-selected.

{/* TODO: Add screenshot - bulk-apply-results.jpeg */}

### 5. Apply Selected

Review the matches, adjust selection if needed, then click **Apply Selected** to tag all selected links.

## Tips

- **Better descriptions = better matches** - Be specific about what the tag represents
- **Use Generate** - Let AI create a description based on already-tagged links
- **Review before applying** - Check medium/low confidence matches manually

## Requirements

- AI provider must be configured (OpenAI or Local AI)
- Tag must have an AI description
- Works with saved links only
