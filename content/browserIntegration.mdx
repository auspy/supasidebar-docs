# Browser Integration

SupaSidebar supports 14 different web browsers with two levels of integration.

## Integration Levels

### Basic Integration
Includes copy URL (⌘⇧C), save page (⌘⇧S) shortcuts, and ability to open links in the browser.

### Live Tabs
Real-time browser tab tracking and switching. See [Live Tabs documentation](LIVE_TABS.md) for details.
