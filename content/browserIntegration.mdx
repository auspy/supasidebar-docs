# Browser Integration

SupaSidebar supports 14 different web browsers with two levels of integration.

## Integration Levels

### Basic Integration
Includes copy URL (⌘⇧C), save page (⌘⇧S) shortcuts, and ability to open links in the browser.

### Live Tabs
Real-time browser tab tracking and switching. See [Live Tabs documentation](LIVE_TABS.md) for details.

## Setup

### Required Permissions

Grant SupaSidebar permission to control browsers:

1. Open **System Settings → Privacy & Security → Automation**
2. Find **SupaSidebar** in the list
3. Enable checkboxes for browsers you want to use

For Live Tabs with certain browsers (Comet), you may also need:
- **System Settings → Privacy & Security → Accessibility** → Enable SupaSidebar
