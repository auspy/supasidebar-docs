## Tips for Productivity

### Smart Save & Copy Context Awareness

The **âŒ˜âŒƒS** (Smart Save) and **âŒ˜âŒƒC** (Smart Copy) shortcuts are context-aware:

- **In a Browser**: Save/copy the current page URL
- **In Finder**: Save/copy selected files or folders
- **Anywhere Else**: Defaults to browser behavior

**Learn more:** [Smart Shortcuts â†’](features/smartShortcuts) - Complete guide to context-aware shortcuts

### Customizing Shortcuts

You can customize most global shortcuts in **Preferences â†’ Shortcuts**:

1. Open SupaSidebar preferences
2. Navigate to the **Shortcuts** tab
3. Click on any shortcut to record a new key combination
4. Press the desired key combination
5. Changes are saved automatically

### Disabling Optional Shortcuts

Some shortcut groups can be toggled on/off:

- **Space Navigation Number Shortcuts** (âŒ˜âŒƒ1-9): Enable/disable in **Preferences â†’ Spaces**
- **Pinned Item Shortcuts** (âŒ˜âŒ¥1-9): Enable/disable in **Preferences â†’ General**

This prevents conflicts with other apps that may use the same shortcuts.
